## 步骤

- 安装 Ollama
  - 访问 [Ollama 官网](https://ollama.com/) 下载并安装 Ollama。
  - 安装其他目录(可选)
    - 运行: `OllamaSetup.exe /DIR=D:/develop/ollama`
  - 配置模型安装的位置(可选)
    - 配置一个环境变量 `OLLAMA_MODELS` 指向模型安装目录
      - 例如: `OLLAMA_MODELS=D:\develop\ollama\models`
  - 打开浏览器访问 [地址](http://localhost:11434/) 检查是否安装成功。
- 安装 deepseek-R1 模型(其他直接替换即可)
  - 访问 [Ollama 官网](https://ollama.com/)
  - 搜索 deepseek-R1 模型, 进入详情页, 选择大小 1.5b
  - 复制命令打开命令行进行安装即可 `ollama run deepseek-r1:1.5b`
  - 然后就可以直接问问题了, 但是目前只能在命令行使用
  - 后续使用还是直接运行 `ollama run deepseek-r1:1.5b` 即可
- 安装 docker
  - 运行 docker 程序
- 安装 Open-webui
  - 访问 [Open-webui githunb 仓库](https://github.com/open-webui/open-webui)
  - 下拉找到 `If Ollama is on your computer, use this command:` 复制命令在命令行中执行
    - 例如: `docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.nju.edu.cn/open-webui/open-webui:main`
    - 3000 为本机端口, 可以根据需要修改
    - 然后浏览器访问 [地址](http://localhost:3000/) 创建管理员账户即可使用了
