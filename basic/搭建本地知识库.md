## 步骤

- 安装 Ollama
  - 访问 [Ollama 官网](https://ollama.com/) 下载并安装。
  - 打开浏览器访问 [地址](http://localhost:11434/) 检查是否安装成功。
- 安装 deepseek-R1 模型(其他直接替换即可)
  - 访问 [Ollama 官网](https://ollama.com/)
  - 搜索 deepseek-R1 模型, 进入详情页, 选择大小 1.5b
  - 复制命令打开命令行进行安装即可 `ollama run deepseek-r1:1.5b`
  - 然后就可以直接问问题了, 但是目前只能在命令行使用
  - 后续使用还是直接运行 `ollama run deepseek-r1:1.5b` 即可
- 安装 AnythingLLM
  - 访问 [AnythingLLM 官网](https://anythingllm.com/) 下载并安装。
  - 对话
    - 聊天模式
      - 即可没有在知识库中找到相关问题, 它也会回答不对的内容
    - 查询模式
      - 将对话模式更改此模式, 如果没有找到相关知识, 就直接回复没有找到
  - 上传文档
    - 在当前工作区上传文档, 询问知识库中的相关问题
    - 例如: 可以将 assets 中统计信息文件上传, 然后询问知识库中的相关问题
  - 语言配置
    - 可以在工作区设置中将系统提示词和拒绝响应配置为中文
